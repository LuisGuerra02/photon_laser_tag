# photon_laser_tag
To start the server, run `npm start` in the root directory of this project.

>Working Paths

* `/` - Displays Splash Screen

* `/players` - Displays all players inside of DB

When using player Entry screen, enter a number into the 'Player ID' field 
and if a 'Player Codename' is not displayed then that ID is not yet used and 
you can enter a new Codename for that ID.
